Behaviour Plugins

Behaviours are plugins that, while unusable in the patcher stage, offer much more utility than a module.

Tip

A completed behaviour can be found in the example mod.

Creating a Behaviour

Creating a behaviour is as simple as can be. Just declare a class that inherits from DeliBehaviour. For example:

using Deli.Setup;

public class MyBehaviour : DeliBehaviour
{
}
